# Etch-A-Sketch
A tool where people can use W,A,S,D to move the pointer and draw images.
Key 'W' - Forwards
Key 'S' - Backwards
Key 'D' - Clockwise
Key 'A' - Anti-clockwise
Key 'C' - Clear Screen and draw afresh
